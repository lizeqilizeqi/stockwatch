# Changelog

本文件记录项目的主要变更。

格式参考 Keep a Changelog，并采用语义化版本思路（当前以手工版本号管理）。

## [0.2.0] - 2026-03-24

### Added

- 深色主题专业看盘界面（ECharts 交互图）
- 新增 `api/stock_timeseries.php`，提供分时/日K序列 JSON
- 图表支持分时图/日K图切换、缩放、十字准星
- 新增项目文档 `README.md`
- 新增 `.gitignore`（忽略运行缓存与日志）

### Changed

- 数据源切换到新浪接口族
- 行情接口增加回退策略：实时接口失败时回退 K 线接口
- 图片接口增加回退策略：GIF失败时后端生成 SVG
- 缓存命名改为 `sina_*` 前缀，避免旧缓存污染

### Fixed

- 修复上游 WebXml 停服导致页面空数据的问题
- 修复部分环境下新浪实时接口不可达导致整体失败的问题

## [0.1.0] - 2026-03-24

### Added

- 初始化项目结构（`index.php`、`api/`、`lib/`）
- 基础看盘页面（行情卡片 + 分时图 + 日K图）
- 服务端 1 小时缓存机制
- GitHub 仓库初始化与首次发布
